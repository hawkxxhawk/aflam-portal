# 🎬 R-Movies IPTV Portal

تطبيق ويب ذكي ومشغل IPTV متكامل لمشاهدة القنوات والأفلام والمسلسلات.

## 🚀 استخراج ملف تطبيق الهاتف (APK) تلقائياً 
هذا المستودع مجهّز بأدوات الاستخراج التلقائي. فقط اتبع الخطوات التالية لتحميل نسختك:
1. قم برفع (أو عمل Push) لهذه الملفات إلى مساحتك (Repository) في حساب Github الخاص بك.
2. افتح تبويب **Actions** في أعلى صفحة مشروعك، ستجد أنه بدأ تلقائياً بعمل Build للتطبيق.
3. بعد الانتهاء (حوالي 3-5 دقائق)، قم بفتح الـ Workflow المكتمل، وفي أسفل الصفحة ستظهر لك أيقونة `app-debug.apk` أو `app-release.apk` في قسم الـ **Artifacts** جاهزة للتحميل الفوري على هاتفك أو تلفازك.

## ✨ المميزات المدرجة
- مشغل ذكي لروابط `M3U` و `Xtream Codes` (يفرز القنوات/الأفلام/المسلسلات تلقائياً).
- واجهة سلسة مع نظام اختصارات سريع لأبرز المواقع والخدمات.
- اختيار لتشغيل البث محلياً أو خارجياً عبر (VLC & PotPlayer).
- ترتيب التصنيفات وقوائم تشغيل مخصصة لحفظ المحتوى وتعديل المجلدات.
- تصميم عصري بلمسات نيون متفاعلة.

**ملاحظة:** التطبيق مبني بلغات الويب الأساسية (`HTML`, `JS`, `CSS`) ويستخدم `Capacitor` للتغليف لنظام أندرويد.
